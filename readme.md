目的
===================================================
* 本プログラムを通してLaravel5.2のCRUDを理解する 

機能(RESTful対応)
===================================================  
* 一覧表示
* 新規作成
* 新規保存
* 表示
* 編集
* 更新
* 削除

内容
===================================================  
* フォームに名前を入力する 
* 入力された名前はDBに保存され表示・編集・削除ができる
